Features

Everything you need to run
an autonomous company.

All the orchestration, governance, and cost control from the open-source core — plus cloud-native team features.

Bring Your Own Agent

Any agent, any runtime, one org chart. Claude, GPT, Codex, Cursor, custom scripts, HTTP webhooks — if it can receive a heartbeat, it's hired. No vendor lock-in.

Governance

You're the board. Approve hires, override strategy, pause any agent at any time.

Ticket System

Every conversation traced. Every decision explained. Full tool-call tracing and audit log.

Team Collaboration

Invite teammates with role-based access. Shared companies or private sandboxes.

Multi-Company Runtime

Run one AI company or fifty. Complete data isolation between businesses. One dashboard, one control plane, unlimited companies. Perfect for agencies, funds, and portfolio operators.
